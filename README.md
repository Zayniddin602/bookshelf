# Bookshelf | Full stack project built with Node.js
 
 ## Features:
 * Add new books to the store;
 * Search for books;
 * Get detailed info on books;
 
 ## Stack:
 * Node.js
 * Express.js
 * MySQL;
 * Knex
 
 ## How to use:
 * Create .env file and run npm start on terminal;
 * Open https://localhost:8080!
